# alexdebiasio.com

This is a small static personal site built with plain HTML and CSS. It does not use Astro, JavaScript, a package manager, or a build step.

## Structure

- `index.html`: home page
- `projects/index.html`: projects page
- `cv/index.html`: CV page
- `styles.css`: shared styles
- `images/`: image assets
- `files/`: downloadable PDFs

## Local Preview

Open `index.html` in a browser.

## Deployment

Deploy the repository root as a static site. There is no production build command.
